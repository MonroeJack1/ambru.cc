# AMBRU.CC Consulting Website

A premium dark-first personal consulting website for **AMBRU.CC**, the professional brand of Cosmin Claudiu.

## Public GitHub Pages URL

The site is configured for the GitHub Pages project URL:

https://monroejack1.github.io/ambru.cc/

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS v4
- shadcn/ui-style local components
- lucide-react icons

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Next.js.

## Build

```bash
npm run build
npm run start
```

## Important customization points

- Replace the CTA email links in `src/app/page.tsx` with a live scheduling link, CRM form, or real inbox.
- Update metadata in `src/app/layout.tsx` once final Open Graph assets are available.
- The founder visual intentionally uses an initials/abstract card, not a fake portrait.
